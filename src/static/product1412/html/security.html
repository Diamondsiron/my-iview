项目安全保障

风控安全卫士

超百人专家团队用心把关，7大环节，层层筛选，多重渠道交叉验证，审批、评级后随机风控稽查，帮助出借人控制风险

反欺诈系统

与同盾等多家知名第三方大数据风控服务提供商合作，利用先进的欺诈风险识别技术，从多维度、频度、设备网络环境、用户行为分析、 全局联防联控信息等方面，识别借款个体的信用风险

嘉业云系统

嘉e贷自主研发的第三代大数据系统，基于大数据分析，精准评估借款人逾期风险

交易安全保障

江西银行存管

嘉e贷联手江西银行，对平台、出借人、借款人存管账户进行开设与管理。交易过程中，借贷双方的每笔资金均在江西银行账户体系内完成，资金流水清晰可查， 平台与用户资金完全隔离，资金交易均由用户授权江西银行操作执行，确保交易真实有效

合同签章认证

利用第三方合作机构提供的电子签章及第三方存证服务，将电子签名制度落实到每笔交易中，保障交易数据的真实性、完整性及电子合同的法律效力

技术安全保障

信息加密

数字证书加密，安全套接协议，非对称加密，保障用户信息及数据安全

容灾备份

生产、容灾两个数据中心同步，物理机与云服务器互为备份，确保数据的完整可靠，历史数据永久保存

内控流程

用户所有个人信息都经过MD5加密处理，内部严格执行信息分级权限控制，所有查询数据都会被记录，保障用户信息安全

法律合规保障

法律合规

平台所有的项目都有实际的个人借款需求，披露的信息都真实有效。项目在借贷关系合法性、利息收益合法性、居间撮合服务合法性及电子合同合法性方面都严格遵守国家相关法律规定

战略合作

嘉e贷聘用君辉律师事务所作为公司的常年法律顾问。平台的法务团队将借助君辉律师事务所专业的金融法律服务为出借人保驾护航
